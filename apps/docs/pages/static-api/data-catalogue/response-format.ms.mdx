## Format Respons

### Respons yang sah

Apabila anda membuat permintaan yang berjaya kepada API Katalog Data, anda akan menerima respons dengan kod status 200. Inilah struktur format respons:

```JSON
{
    meta: {...},
    data: {...}
}
```

- `meta` - Medan ini menunjukkan maklumat meta ringan tentang sumber yang diminta.
- `data` - Medan ini mewakili sumber yang diminta, yang mungkin mengandungi maklumat tambahan berdasarkan parameter kueri yang diberikan.

### Respons Ralat

Dalam kes ralat, API Katalog Data akan mengembalikan respons ralat dalam format JSON. Respons ralat akan mengandungi unsur-unsur berikut:

```json
{
    status: <int>,
    errors: [...],
}
```

- `status` - Medan ini menunjukkan kod Respons yang sepadan dengan ralat.
- `errors` - Medan ini menyediakan rangkaian mesej atau keterangan ralat.

Respons ralat direka untuk membantu anda mengenal pasti dan menyelesaikan masalah yang anda hadapi dengan permintaan API anda..
